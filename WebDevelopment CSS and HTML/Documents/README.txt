# 📚 Gatsheni Online Book Store
Turn the page, change your life.

## 🚀 About the Project
This is an online bookstore website created using **HTML** and **CSS** only.  
It allows users to browse books, view categories, and manage a shopping cart.

## 📂 Project Structure
- `index.html` → Home page
- `categories.html` → Book categories
- `books.html` → All books
- `cart.html` → Shopping cart
- `about.html` → About the store
- `CSS folder` → Contains all the stylesheet

## ✨ Features
- Responsive navigation bar with logo and menu
- Categories section with book cards
- Add to cart functionality (UI design)
- Simple, clean design with consistent layout

## 🛠️ Built With
- HTML5  
- CSS3
